Getting Started

Welcome to AI Storyboard! This tool allows you to generate visual storyboards from text scripts using the power of Large Language Models (LLMs) and image generation APIs.

security

Important: Privacy & Security

Your API keys are stored locally in your browser to communicate directly with LLM and Instagram services. We do not store your keys on our servers. Clearing your browser cache will remove these keys.

key

API Configuration

To use the generator, you need to provide your own API keys. This allows you to control your usage and costs directly with the providers.

smart_toy

1. LLM Provider

OpenAI or Anthropic for script analysis.

  1. Sign up at OpenAI Platform or Anthropic Console.
  2. Navigate to the API Keys section in your dashboard.
  3. Create a new secret key (starts with sk-...).
  4. Copy the key immediately.
sk-proj-1234567890abcdef...
photo_camera

2. Instagram Graph API

For publishing or fetching media assets.

  1. Go to Meta for Developers and create an app.
  2. Add "Instagram Graph API" product.
  3. Generate a User Access Token via Graph API Explorer.
  4. Ensure `instagram_basic` permission is checked.

Using the Storyboarder

1

Input Your Script

Paste your video script, dialogue, or rough notes into the main text area on the Home page. The AI works best with descriptive scene settings.

Good Prompt Example

"Exterior, Day. A busy futuristic street market in Tokyo. Neon signs glow even in daylight. A young woman with blue hair buys an apple from a robot vendor."

2

Generate Scenes

Click the Generate button. The system will first break your script into scenes using the LLM, then generate an image for each scene using the image generation API. This process usually takes 30-60 seconds.

3

Edit & Export

Review the generated images. You can regenerate specific panels if they don't match your vision. Once satisfied, export the entire board as a PDF or ZIP file.

FAQ & Troubleshooting

Is this application free to use? expand_more

The web application itself is free to use. However, because it relies on your own API keys, you will be billed directly by the API providers (e.g., OpenAI, Anthropic) based on your usage.

Where are my storyboards saved? expand_more

All storyboards and project data are saved locally in your browser's storage (IndexedDB/Local Storage). We do not persist your creative work on our servers. Make sure to export your work before clearing browser data.

Which LLM model works best for storyboarding? expand_more

For detailed scene descriptions and complex narratives, we recommend using stronger models like GPT-4 or Claude 3 Opus. For faster, more cost-effective results, GPT-3.5 Turbo or Claude 3 Haiku are sufficient.

"Invalid API Key" Error expand_more

Double-check that you copied the entire key string without extra spaces. For Instagram keys, ensure the token hasn't expired (standard tokens expire in 1 hour; generate a long-lived token for better experience).

Images are not generating expand_more

This is often due to a rate limit from the provider or insufficient credits in your OpenAI/Anthropic account. Check your usage dashboard on the provider's website.

How do I clear my keys? expand_more

You can clear your keys by clicking the "Reset Settings" button in the Settings page, or by manually clearing your browser's Local Storage for this site.
